Цель:
разработать доступную и интуитивно понятную платформу для потокового
воспроизведения музыки, которая позволит пользователям слушать,
сохранять и делиться музыкальными произведениями. Платформа должна
поддерживать взаимодействие пользователей, создание плейлистов 
и возможность оценки музыки.
